# Подсказки по командной строке

Команда смена директории:
```sh
cd c:\folder_name
```
Команда отображения текущей директории MacOs, Linux:
```sh
pwd
```
Листинг текущей директории:
1. Windows:
```sh
dir
```
2. Linux, MacOs:
```sh
ls
```
Удаление файла:
1. Windows:
```sh
del <filename>
```
2. Linux, MacOs:
```sh
rm <filename>
```
